My Classes at OCC

PHI 1710 – Informal Logic

Informal Logic is a class that focuses on reasoning and argument in modern life. The class discusses topics such as Understanding arguments, the types of arguments, fallacies, critical thinking, and more.


CIS 2111 – System Analysis & Design

System Analysis & Design is a class focused on how systems function, efficient use of team management, and designing UML diagrams. The class includes topics like Systems Development Life Cycle, UML diagrams, and Team management methodologies like Agile and Scrum.


CIS 1420 – HTML5 Programming

HTML5 Programming is a class for all aspects of front-end development. The class demonstrates JavaScript, CSS, and HTML. The class also implements real-world use of the topics for hosting the website students created.
